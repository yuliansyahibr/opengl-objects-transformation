### Objects:

### Controls:

ESC = exit <br>
Spasi = menu (change background color and object selection) <br>
<br>
W, A, S, D = camera <br>
mouse = camera <br>
<br>
Arrow = translate object <br>
Q + arrow = rotate object <br>
Z = Enlarge objek <br>
X = shrink objek <br>
